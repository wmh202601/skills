# Claw CAMP 海报类型规范

## 两种海报，用途不同

| 类型 | 用途 | 比例 | 尺寸 |
|------|------|------|------|
| **首发海报**（短） | 微信群/朋友圈首次传播，一屏完整显示 | 9:16 竖屏 | 420 × 747px |
| **详情长图** | 感兴趣后发的详情版，可滚动 | 无限长（max-width: 420px） | 420 × 不限 |

---

## 类型一：首发海报（9:16）

**核心原则**：一屏完整，不需要滚动，10 秒内决定是否感兴趣。

### 固定尺寸

```css
.poster {
    width: 420px;
    height: 747px;
    overflow: hidden;
    display: flex;
    flex-direction: column;
}
```

### 必须包含的模块（按顺序）

1. **彩条** — 5px，品牌规范见 brand.md
2. **Header（白底）**
   - 顶部导航：品牌 + 期次 badge + Powered by
   - 主标题（1.75–2rem，800–900 weight）
   - 金句/slogan（1–2 行）
   - 信息 chips：日期 · 地点 · 席位（橙色 hot chip）
3. **席位进度条** — 视觉紧迫感（已满 X/12）
4. **三列收获卡片** — 绿/橙/蓝三色，各一个核心交付物
5. **数字亮点行** — 4 个关键数字，竖分隔线
6. **无需门槛行** — ✗ 不用代码 · ✗ 不用技术 · ✗ 带电脑和手机
7. **底部 CTA（深色）** — 扫码锁席位 + 二维码 + 退款承诺

### 高度预算参考

```
color-bar:    5px
header:      ~270px（含 padding）
urgency:      36px
benefits:    130px
stats:        56px
prereqs:      28px
footer-cta:  ~222px（flex: 1，自适应剩余空间）
总计:         747px
```

### 文案原则

- **标题**：有画面感，让人一眼看懂"我能得到什么"
- **金句**（当前确认版本）：
  > 带一个真实业务场景来 **→** 带一套 AI 解法走
- **受众行**：不要身份标签列表，用状态/行为描述代替
- **数字行**：展示用户关心的结果（2天 / 带走即用 / 1v1 / 全额退），不展示课程结构（如"10模块"）
- **席位**：`仅剩 X 席` + 进度条（已满 8/12）

---

## 类型二：详情长图

**核心原则**：信息完整，适合感兴趣后深度阅读，可以滚动。

### 结构

```css
body {
    max-width: 420px;
    margin: 0 auto;
    /* 高度不限，允许滚动 */
}
```

### 包含模块（可含）

1. 彩条 + Header（同首发）
2. 痛点区（真实案例 · 3条）
3. 核心价值区（带走什么）
4. 日程模块列表（DAY1 / DAY2 · 详细）
5. 数据背书（GitHub Stars / 平台接入数 / Skill数量）
6. 适合谁（目标人群）
7. 无需门槛
8. 导师介绍
9. 底部 CTA（含或不含价格，按需求）

### 价格处理

- **无价格版**：删除 `.price-row` 整块，保留 QR + CTA 文字
- **有价格版**：原价划线 + 现价 + 备注（早鸟/含物料）

## 渠道 QR 替换规范

海报底部二维码按分发渠道替换：

```html
<!-- 按渠道替换为对应伙伴的二维码图片 -->
<img src="梦浩微信.jpg" alt="扫码咨询" class="qr-img">
```

- **通用/默认**：`梦浩微信.jpg`
- **Jenny 渠道**：`Jenny 微信.jpg`
- **Angel 渠道**：`angel 微信.jpg`
- **其他伙伴**：放入 ClawCAMP 文件夹，命名为 `XXX微信.jpg`

图片样式：`width: 100–120px; height: auto;`（不要固定高度，保持原始比例）
